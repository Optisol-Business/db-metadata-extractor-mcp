# db-metadata-extractor-mcp

A Python MCP stdio server for database metadata extraction.

## Installation

```bash
pip install db-metadata-extractor-mcp
```

## Usage

```bash
db-metadata-extractor-mcp
```

## Features
- Extracts database metadata
- Provides a Model Context Protocol (MCP) stdio server
